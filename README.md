### Features:
1. RSS feed.
2. Share via email.
3. Filter posts by tags.
4. Supports Markdown.
5. View the latest/most commented posts and the ones with popular tags.

### Technology stack:
- Framework: Django
- Database: SQLite
- Hosted on: Heroku
